BalanceSigns

Economy Bukkit plugin for Minecraft 1.8.8

Overview

Compiled for Minecraft version 1.8.8

Requires Vault for economy rewards (Tested with Vault 1.5.5)

Put [topbalance] on a sign to get started

Wait for server refresh

Right click on sign opens GUI with top players

Features

  • Returns top players from server with highest amount of ecoconmic worth.
  • GUI Based
  • Adjustable config to change paramenters of how the plugin works
  • No permissions
